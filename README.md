1. Clone the repository:  
   ```bash  
   git clone https://github.com/yourusername/weather-app.git  
   ```  

2. Navigate to the project directory:  
   ```bash  
   cd weather-app  
   ```  

3. Open `index.html` in your browser to view the app.
